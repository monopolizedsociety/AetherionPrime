# Frontend (React)
This folder will host the React app (MetaMask Embedded Wallet UI, onboarding flows, dashboards).

## Suggested stack
- React + Vite
- Ethers.js
- MetaMask Embedded Wallet SDK
- Solana Web3.js

## Quick start (example with Vite)
npm create vite@latest . -- --template react
npm install
npm run dev
